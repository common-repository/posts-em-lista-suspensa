Plugin que lista todos os posts de uma determinada categoria numa caixa dropdown (lista suspensa) em uma outra p&aacute;gina ou post - ou seja, cria uma esp&eacute;cie de categoria numa p&aacute;gina ou post -, personalizavel e bem simples de trabalhar.

== Description ==

* Cria links para posts em lista suspensa automaticamente;
* Pode ser personalizado;
* Com ou sem bot&atilde;o de confirma&ccedil;&atilde;o;
* Inser&ccedil;&atilde;o pelo post, p&aacute;gina ou template PHP.

== Installation ==

* 1 - Fa&ccedil;a o download do plugin;
* 2 - Efetue upload do zip para o servidor na pasta de plugins;
* 3 - Ative o plugin no painel de controle;
* 4 - Configure-o em op&ccedil;&otilde;es.
